## üìù Description

**EZPassOH** is a user-friendly application designed to simplify the tracking and management of E-ZPass toll expenses, specifically catering to users of the Ohio Turnpike and the broader E-ZPass network. The official E-ZPass system provides electronic toll collection, offering users an average of 33% savings on Ohio Turnpike tolls, reduced travel time, and increased convenience by allowing non-stop passage through dedicated lanes. This application aims to complement these benefits by offering a local interface for users to manually log, categorize, and calculate their travel costs, generate simple receipt-like summaries for personal or business records, and visualize spending habits. It's built with React, ensuring a responsive and modern user experience. The primary goal is to provide a fast, offline-capable tool for expense aggregation and reporting.

## ‚ú® Features

* **Toll Logging:** Easily record individual toll transactions, including date, location, and cost.
* **Expense Summarization:** Generate quick summaries of total spending over a specified period.
* **Receipt Generation:** Create simple, formatted digital "receipts" for recorded expenses.
* **List Management:** View and manage a list of all recorded toll transactions.
* **User Interface:** Clean, component-based structure built with React for a dynamic experience.
